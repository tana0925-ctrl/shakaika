# 社会科同好会 成長の道しるべ

## Project Overview
- **Name**: 社会科同好会 成長の道しるべ（学びのコンパス）
- **Goal**: 名古屋社会科同好会のメンバーが、自分の現在地と次のステップを把握できるルーブリック型成長マップを提供する
- **Features**: 
  - 5つの成長視点（授業をつくる・授業をする・子供を見る・つながる・深める）
  - 4段階のステップ（まずはここから → 自分で工夫する → みんなと高める → 未来を創る）
  - 各ステップに対応するおすすめアクション
  - A4横印刷に最適化されたレイアウト

## URLs
- **Preview**: ローカル開発サーバーで確認可能
- **API**: `/api/health` - ヘルスチェックエンドポイント

## Data Architecture
- **Storage**: 静的コンテンツのみ（データベース不要）
- **Rendering**: サーバーサイドHTML生成（Hono `c.html()`）

## User Guide
1. ページにアクセスすると、成長ロードマップの表が表示されます
2. 3つのカテゴリ（授業準備・仲間活動・研究発信）に分かれた5つの視点で確認できます
3. 各セルにはキーワードと具体的な行動指針が記載されています
4. 印刷ボタン（Ctrl+P）でA4横1枚に印刷できます
